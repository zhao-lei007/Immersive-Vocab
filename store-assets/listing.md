## 基本信息

- **名称**：Immersive Vocab - 沉浸式背单词
- **类别**：教育（Education）
- **语言**：中文（简体）

### 摘要（Summary，≤132 字符）

```
沉浸式网页翻译 + 生词本：划词查词一键收藏，音标释义例句自动补全，闪卡间隔复习。支持本地/自定义 AI 模型，数据全部保存在本地。
```

### 详细描述（Description）

```
在阅读英文网页的同时积累词汇——这就是沉浸式背单词做的事。

【翻译】
• 全文双语对照翻译，原文译文逐段对照
• 划词即时翻译，右键菜单快捷入口
• 支持 OpenAI 兼容接口：本地模型（Ollama/LM Studio/MLX 等）、DeepSeek、通义千问等均可接入，也支持微软/谷歌免费翻译

【查词与收藏】
• 词典弹窗：音标、词性、释义、例句即点即得
• 一键收藏到生词本；整句划选时可从原文单独挑词收藏
• 收藏即入库，译文、音标、例句由 AI 后台自动补全

【复习】
• 侧边栏生词本：闪卡复习，答对间隔翻倍，答错稍后重现（简化间隔重复）
• 翻译弹窗内嵌到期词快速复习
• 已收藏的词在网页上自动高亮，随处巩固

【数据属于你】
• 生词本保存在浏览器本地，不经过任何第三方服务器
• 支持导出/导入 JSON 备份
• 可选：绑定本地文件夹（iCloud Drive/坚果云等网盘目录），用你自己的网盘跨设备同步

本扩展基于开源项目 read-frog（GPL-3.0）二次开发，源代码公开：
https://github.com/zhao-lei007/Immersive-Vocab
```

## 隐私相关（Privacy 标签页）

### 单一用途说明（Single purpose）

```
帮助用户在阅读网页时翻译内容、查询并收藏生词，通过间隔重复复习积累词汇。
```

### 权限说明（Permission justifications）

| 权限               | 理由（可直接粘贴）                                                       |
| ------------------ | ------------------------------------------------------------------------ |
| `storage`          | 在本地保存扩展设置与生词本数据                                           |
| `tabs`             | 读取当前标签页地址与状态，用于按站点开关翻译、打开侧边栏生词本           |
| `alarms`           | 定期清理过期的翻译缓存                                                   |
| `contextMenus`     | 提供右键菜单翻译/朗读/自定义 AI 动作入口                                 |
| `identity`         | 可选的 Google Drive 配置同步功能，OAuth 授权仅访问用户自己的应用数据目录 |
| `scripting`        | 向页面注入翻译、划词工具条与生词高亮脚本                                 |
| `webNavigation`    | 侦测页面与 iframe 导航，在单页应用切换后保持翻译状态                     |
| `offscreen`        | 在后台播放单词发音（TTS 音频）                                           |
| `sidePanel`        | 承载生词本侧边栏（列表、闪卡复习、同步设置）                             |
| 主机权限 `*://*/*` | 翻译与生词高亮需要在用户访问的任意网页上工作                             |

### 数据使用声明（Data usage）

- 全部勾选"**不收集**"：本扩展不收集、不传输任何用户数据
- 生词本与设置仅存储在浏览器本地（IndexedDB / storage）
- 翻译请求直接从浏览器发往用户自己配置的 AI/翻译服务，不经过开发者服务器
- 远程代码：无（所有代码打包在扩展内）

## 上架步骤

1. 注册开发者账号：https://chrome.google.com/webstore/devconsole 用 Google 账号登录，支付一次性 $5 注册费
2. "新增项目" → 上传 `.output/immersive-vocabextension-1.37.5-chrome.zip`
3. 商店资讯页：粘贴上面的名称/摘要/描述，上传 3 张截图，类别选"教育"
4. 隐私页：粘贴单一用途、逐项粘贴权限理由、数据使用全选"不收集"、填隐私政策 URL
5. 发布设置：公开（Public）
6. 提交审核（首次审核通常 1~3 个工作日；因为有 `*://*/*` 主机权限，可能更久）

## 审核注意事项

- 名称/描述避免出现 "Chrome"、"Google" 字样（政策限制）
- 若审核被拒最常见原因是"权限用途不明"——上面的权限理由已按功能一一对应
- 上架后更新版本：改 `package.json` version → `pnpm zip` → 后台上传新包
